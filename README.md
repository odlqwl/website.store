**1. Тема и концепция**  

Название: APEX  

Краткое описание:  
APEX - это современный интернет-магазин спортивных товаров, предлагающий широкий ассортимент продукции для активного образа жизни: от одежды и обуви до тренажеров. Платформа помогает спортсменам и любителям находить качественную экипировку, инвентарь и аксессуары в удобном цифровом формате. Проект обеспечивает быстрый и удобный доступ к товарам ведущих мировых брендов. 

Целевая аудитория:  
- Профессиональные спортсмены - люди, которым требуется специализированная экипировка и инвентарь для тренировок и соревнований.
- Любители фитнеса и активного образа жизни.  
- Покупатели подарков.  
- Родители - приобретающие спортивные товары для детей (секции, школы, активный отдых).

Ключевой функционал:  
- Система аутентификации — регистрация, вход, восстановление пароля.  
- Каталог товаров — с фильтрацией по категориям и ценовому диапазону.  
- Детальная страница товара — фото, описание, характеристики, цена.  
- Система поиска.  
- Управление корзиной — добавление, удаление товаров.  
- Оформление заказа — ввод адреса доставки, выбор способа оплаты.  
- Личный кабинет — просмотр и редактирование профиля.  
- Административная панель — управление товарами, заказами и пользователями.

**2. Структура приложения**  
1) Главная (/) – публичная  
2) Аутентификация (/auth/*) – публичная
3) Каталог (/catalog) – публичный, динамический список  
4) Страница товара (/catalog/:id) – динамическая, публичная    
5) Личный кабинет (/profile) – приватный  
6) Оформление заказа (/order) – приватный
7) Корзина (/cart) – приватная 
8) Админ-панель (/admin) – приватный (опциональная)

**3. Технологический стек** 

  *Frontend:*  
  
  - Фреймворк: Vue 3  
  Выбран, потому что он простой и понятный, что позволит быстро создать интернет-магазин спорттоваров. Страницы будут загружаться быстро, а покупателям будет удобно листать каталог и смотреть товары. Благодаря гибкости фреймворка, в магазин легко добавлять новые возможности, например, умные фильтры или корзину.  
  - Роутер: Vue Router  
  Выбран за высокую производительность, гибкую систему реактивности и удобную структуру кода через Composition API. Отвечает за навигацию по сайту — переходы между главной страницей, каталогом, карточками товаров и личным кабинетом.  
  - State-менеджмент: Pinia  
  Это лёгкое и современное средство для хранения и управления общими данными приложения. С его помощью удобно отслеживать состояние корзины, данные авторизации и выбранные фильтры в каталоге, а благодаря поддержке Composition API и типизации код становится понятнее и надёжнее.  
  
  *Backend:*  
  
  - Среда разработки: Node.js  
Позволяет использовать JavaScript на стороне сервера, упрощает передачу данных между фронтендом и бэкендом. Хорошо справляется с большой нагрузкой, когда много покупателей одновременно ищут товары или оформляют заказы.  
  - Фреймворк: Express.js  
Лёгкий и гибкий фреймворк, который обеспечивает удобную маршрутизацию и обработку запросов. Позволяет быстро создать REST API для работы с товарами, пользователями и заказами, а также реализовать проверку прав доступа.  
  - База данных: Supabase (PostgreSQL)  
Надёжное облачное решение на основе PostgreSQL с встроенной системой аутентификации и автоматическим созданием API. Обеспечивает быстрое выполнение сложных запросов для фильтрации товаров по множеству параметров, безопасное хранение пользовательских данных и удобное управление изображениями товаров.

  *Инфраструктура:* 

  - Хостинг Frontend: GitHub Pages  
Бесплатный и простой способ публикации клиентской части приложения. Автоматически обновляет сайт при изменениях в репозитории и обеспечивает безопасное соединение с пользователями.  
  - Хостинг Backend: localhost
